Project Overview

Aria and the Lost Souls is a 2D adventure game that combines platforming mechanics with puzzle-solving elements. Players embark on a mystical journey through beautifully crafted environments.

Shards of Ylena

3D Action-Adventure Prototype

Project Overview

Shards of Ylena is a solo action-adventure game prototype in which the player takes on the role of an agile archer.
The development has so far focused on building the core player gameplay, including fluid movement mechanics (running, jumping, dodging) and a responsive archery system at the heart of the experience.
This project lays the groundwork for a dynamic and strategic gameplay loop within a fantasy world still in early development. It serves as a technical base to experiment with character control, shooting feel, and player responsiveness, ahead of future additions like enemies, special abilities, or an explorable environment.

Key Features

  • Prototype of a 3D action-adventure game built in Unreal Engine
  • Core player movement system: walking, jumping, dodging
  • Bow shooting mechanics with basic projectile handling
  • Third-person camera system tailored for ranged combat
  • Basic environment elements for testing gameplay responsiveness
  • Initial groundwork for future abilities (elemental arrows, skills)
  • Early-stage animations using AnimBlueprint for movement and aiming
